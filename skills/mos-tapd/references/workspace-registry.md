## 平台中心（61406540）额外配置

### 实体类型说明

平台中心**没有独立的 tasks 模块**（API 返回 403）。所有"任务"通过**需求（story）的子类别**实现：

| 类别名 | workitem_type_id | 适用场景 |
|--------|-----------------|---------|
| 需求 | 1161406540001000087 | 父需求 |
| 产品任务 | 1161406540001000149 | 产品相关 |
| 设计任务 | 1161406540001000150 | 设计相关 |
| 后端任务 | 1161406540001000151 | 后端开发 |
| 数开任务 | 1161406540001000152 | 数据开发 |
| 前端任务 | 1161406540001000153 | 前端开发 |
| 客户端任务 | 1161406540001000154 | 客户端开发 |
| 测试任务 | 1161406540001000155 | 测试相关 |
| 运维任务 | 1161406540001000156 | 运维/基础设施 |
| 通用任务 | 1161406540001000157 | 其他/默认 |

> **选择规则**：根据任务描述推断最匹配的类别，无法判断时使用"通用任务"。

### 各类别状态工作流

#### 需求状态流

未开始 → 已确认 → 规划中 → 设计中（可跳过） → 研发中 → 测试中 → 验收中 → 待发布 → 已上线 | 已取消 | 已阻塞

#### 研发任务流（后端/数开/前端/客户端共用）

未开始 → HLD梳理中（可跳过） → 待HLD评审 → HLD调整中（可跳过） → 研发中 → 联调中 → 待测试 → 测试中 → 已完成 | 已取消 | 已阻塞

#### 产品任务流

未开始 → 产品调研中（可跳过） → 产品设计中 → 待业务确认 → 产品修改中（可跳过） → PRD梳理中 → 待需求评审 → PRD调整中（可跳过） → 已完成 | 已取消 | 已阻塞

#### 设计任务流

未开始 → 设计调研中（可跳过） → 设计中 → 待业务确认 → 设计修改中（可跳过） → 待设计评审 → 设计调整中（可跳过） → 待走查 → 走查中 → 研发调整中 → 已完成 | 已取消 | 已阻塞

#### 测试任务流

未开始 → 用例梳理中（可跳过） → 待用例评审 → 用例调整中（可跳过） → 待测试 → 测试中 → 已完成 | 已取消 | 已阻塞

#### 运维任务流

未开始 → 方案制定中（可跳过） → 待方案评审 → 方案修改中（可跳过） → 待执行 → 验证中 → 方案调整中（可跳过） → 已完成 | 已取消 | 已阻塞

#### 通用任务流

未开始 → 进行中 → 已完成 | 已取消 | 已阻塞

#### 缺陷状态流

待确认 → 已确认 → 修复中 → 待验证 → 已关闭 | 重新打开 → 已取消 | 重复缺陷 | 无需处理
