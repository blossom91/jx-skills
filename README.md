# jx-skills

公开的 Claude Code Skills 集合，解决内网 GitLab 权限限制问题。

## 安装

```bash
npx skills add blossom91/jx-skills
```

安装指定 skill：

```bash
npx skills add blossom91/jx-skills --skill mos-core
```

## 包含的 Skills

| Skill | 说明 |
|-------|------|
| mos-core | mos CLI 核心操作指南 |
| mos-gos | mos CLI GOS（游戏运营）操作 |
| mos-bi | mos CLI BI（大数据）操作 |
| mos-tapd | TAPD 项目管理操作 |

## 更新

```bash
npx skills update
```
